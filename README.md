# CoreWeb Resmi Web Sitesi (Public)

CoreWeb'in özel tasarım, yönetim paneli, hafif mimari ve SEO odaklı teknoloji çözümlerini anlatan resmi web sitesi.

## Özellikler

- **Sıfır Bağımlılık**: Vanilla HTML/CSS/JS.
- **Sistem Fontları**: Maksimum performans ve yükleme hızı.
- **SaaS / Teknoloji Tasarımı**: Sade ve premium Apple estetiği.
- **Güvenli Form Altyapısı**: Spam korumalı form yönetimi.
- **Vercel Edge Dağıtımı**: Küresel ölçekte hızlı ve kesintisiz yayın.

## Geliştirme

Yerel sunucuyu başlatmak için:
```bash
npm install
npm run dev
```

Üretim derlemesi oluşturmak için:
```bash
npm run build
```
